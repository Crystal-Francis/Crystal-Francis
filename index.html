About Me



  I am Crystal Francis, an 11-year-old girl studying at Jing shan Primary.
  You are viewing my Portfolio I created by myself using HTML and CSS.


   Python

  I started learning Python in June 2021. My brother introduced it to me as he was also
  studying it for his computing class. After watching a tutorial, I created my very first
  Python program. The program would find the user's age when the birth year was given.
  I then started creating projects using math topics such as finding the area of a sphere
  when the radius was specified. Afterward, I created the program that I was the proudest of,
  it would determine the HCF or LCM of two numbers.


   Linux

  After getting my new computer in April 2022, my brother had the idea of booting Linux as my
  operating system (OS). After doing so, I went online to learn basic Linux commands.


   Web Development

  I figured to get into a secondary school via DSA. I would have to make a portfolio. While,
  I searched for websites to create portfolios. I stumbled upon
  a revolutionary web creation platform called EditorX.
  After reading about EditorX online, I discovered that it was launched in February 2020 by Wix,
  a Web development company. Afterward, I searched Web Development and learned that it's the work
  that happens behind the scenes to make a website look good. To be a web developer, I would need to study these
  3 main languages, JavaScript, a programming language, HTML, a markup language,
  and CSS, a style sheet language. I decided to make a portfolio using these three languages.


   JavaScript

  In May 2022, I completed the JavaScript course on Codecademy, a platform that offers free
  coding classes. I did not have Pro or Pro Lite access to receive a,
  certificate of completion. I used JavaScript for my Payment details and Ecommerce store.
  Which also included PHP, SQL, HTML, and CSS. I have only finished one course out of 47 of them in JavaScript.
  Around December 2022. I learnt jQuery, a JavaScript framework, to add effects to my projects.


   HTML

  After finishing part of the HTML course in Codecademy, I went on to download Visual Studio Code (vscode)
  to create my portfolio. And when I was not sure about certain tags, I would look them up on the web
  and learn something new. I then finished the HTML course in July 2022.


   CSS

  I learned CSS in the process of learning HTML. When creating my portfolio, I wanted to style my page.
  After looking it up, I learned about CSS properties. I then completed the CSS course in August 2022.
  In November, I learnt Bootstrap, a CSS framework and received a certificate of completion from Codecademy.
